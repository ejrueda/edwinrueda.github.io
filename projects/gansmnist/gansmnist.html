Generative Adversarial Networks (GANs) for MNIST

Se crea una arquitectura GAN para generar nuevos dígitos escritos a mano a partir de una distribución normal, consiguiendo así poder aumentar el conjunto de datos para posteriormente poder ser utilizados en otro tipo de tareas de AI.

Para validar la capacidad de generación de dicha arquitectura y para entrenar el Discriminador de dicha arquitectura, se utiliza el banco de datos MNIST el cual contiene 60.000 imágenes escritas a mano con una resolución de (28,28,1) en escala de gris.

La imágen que se muestra representa los dígitos generados por la red generadora de la GAN, teniendo en cuenta que estos dígitos son generados a partir de un vector de 200 características, el cual sigue una distribución normal.
